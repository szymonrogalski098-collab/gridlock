# Gridlock

A mobile-first neon cyberpunk browser game. Dodge lasers and falling blocks on a 16×16 grid.

## How to play

Open `cube_master.html` in a browser. No build step or server required.

- **Dash** — click any cell within 5 tiles to move your cube
- **Lasers** — dim red = charging, bright red = firing. Don't be on a laser row or column when it fires
- **Blocks** — dim purple = incoming. Move before a block lands on your cell
- **Coins** — earn 1 coin per survived round (3 in Hard Mode)
